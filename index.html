To share data across all team members, host this file online (free, 5 minutes)

⚠️ Why you're seeing this
You opened this file directly from your computer/phone (file://). Firebase blocks this for security. The file must be hosted at a real URL so everyone can open it in Chrome.
📋 Option 1: GitHub Pages (Free, Recommended)
1
Go to github.com → Create free account → Click New repository → Name it iqc-register → Public → Create
2
Click uploading an existing file → Upload IQC_Register_v4.html → Rename it to index.html → Commit
3
Go to Settings → Pages → Branch: main → Save. Your URL will be:
https://yourusername.github.io/iqc-register/
4
Share that URL with all team members. They open it in Chrome on their phone — done! ✅
🔥 Also fix Firebase Database Rules
Go to console.firebase.google.com → Your project → Realtime Database → Rules → Replace with:
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
Click Publish after pasting.
✅ Once hosted: All team members open the same URL in Chrome. Data saves to Firebase. Everyone sees updates in real time. Team IDs persist forever.
